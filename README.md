# Mohamed Alaa | Multi-page Portfolio Website

A responsive multi-page portfolio website built with **HTML**, **CSS**, and **JavaScript** using **Tailwind CSS** as the framework layer.

## Pages
- `index.html` — Home
- `about.html` — About
- `projects.html` — Projects
- `contact.html` — Contact

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- LocalStorage
- FormSubmit (email delivery)
- Intersection Observer API

## Key Features
- Responsive navigation bar
- Dark / light mode
- Mobile menu toggle
- Scroll reveal animations
- Project filtering
- Contact form validation
- Contact form draft autosave with localStorage
- Back-to-top button
- Fully responsive layout for desktop, tablet, and mobile
